Perguntas Frequentes

Quais versões do MS Project são suportadas?

Suportamos arquivos .mpp das versões 2007, 2010, 2013, 2016, 2019 e Microsoft Project 365.

O XML gerado é compatível com outros sistemas?

Sim, geramos XML no formato padrão Microsoft Project Schema, compatível com a maioria dos sistemas de gerenciamento de projetos.

Minhas informações ficam seguras?

Sim, todos os arquivos são processados de forma segura e excluídos automaticamente após 24 horas.
