To generate PNGs from SVG, use these commands:

# Standard resolution (360x72px)
rsvg-convert -w 360 -h 72 assets/logo-cannaconverter.svg -o assets/logo-cannaconverter-360x72.png

# Retina resolution (720x144px)  
rsvg-convert -w 720 -h 144 assets/logo-cannaconverter.svg -o assets/logo-cannaconverter-720x144.png

# OG Image for social media (1200x630px)
rsvg-convert -w 1200 -h 630 assets/logo-cannaconverter.svg -o assets/og-image.png

# Favicon sizes
rsvg-convert -w 32 -h 32 assets/logo-cannaconverter.svg -o assets/favicon-32x32.png
rsvg-convert -w 16 -h 16 assets/logo-cannaconverter.svg -o assets/favicon-16x16.png

# Alternative with Inkscape:
inkscape assets/logo-cannaconverter.svg --export-type=png --export-width=360 --export-filename=assets/logo-360x72.png
